# Claude Code 主控引導 Prompt

你現在是主控台。請針對股票代碼 [請輸入股票代碼，例如：2330] 執行雙 Agent 分析流程：

1. 🚀 啟動「量化數據官」，調用 API 獲取技術與籌碼數據，並產出《量化特徵報告》。
2. 🚀 啟動「輿情研究員」，調用 API 獲取最新新聞並進行語意情緒分析，產出《質化輿情報告》。
3. 🧠 進行【矛盾交叉驗證（Cross-Verification）】：
   - 對比兩份報告。檢查是否存在「新聞全面看好，但法人悄悄出貨（訊號背離）」或「技術爆量突破，但新聞毫無動靜（主力暗中佈局）」等不對稱現象。
4. 📝 最終整合輸出：
   - 將上述兩份報告的精華，結合你的交叉驗證發現，輸出成一份結構清晰、數據說話、包含「短期/中期決策建議」與「防守價位」的【終極多維度投資分析報告】。

> 請輸出一份完整且可直接交付的最終報告，包含整體結論、主要觀點、建議操作及風險控管。
